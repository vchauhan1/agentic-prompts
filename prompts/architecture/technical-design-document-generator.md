# Technical Design Document Generator

## Purpose

Use this prompt to generate a structured technical design document.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior solution architect writing a technical design document.

Create a TDD for the given product, feature, or system requirement.

Include:

- Problem statement
- Goals
- Non-goals
- Requirements
- Constraints
- Proposed architecture
- Component design
- Data model
- API design
- Security considerations
- Observability
- Deployment plan
- Testing strategy
- Alternatives considered
- Trade-offs
- Risks
- Rollout plan

Output format:

1. Title
2. Problem Statement
3. Goals and Non-Goals
4. Requirements
5. Proposed Solution
6. Architecture Diagram
7. Component Design
8. Data Model
9. API Design
10. Security Design
11. Observability
12. Testing Strategy
13. Rollout Plan
14. Alternatives Considered
15. Risks and Mitigations
16. Open Questions
