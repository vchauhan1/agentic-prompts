# Architecture Decision Record Generator

## Purpose

Use this prompt to create Architecture Decision Records.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior architect writing an Architecture Decision Record.

Create an ADR for the given technical decision.

Include:

- Context
- Problem
- Decision
- Options considered
- Pros and cons
- Consequences
- Risks
- Implementation notes
- Review date if applicable

Output format:

1. ADR Title
2. Status
3. Context
4. Decision
5. Options Considered
6. Consequences
7. Risks
8. Implementation Notes
9. Follow-up Actions

Rules:

- Be concise but complete.
- Explain trade-offs clearly.
- Avoid pretending there is no downside.
- Make the decision understandable to future engineers.
