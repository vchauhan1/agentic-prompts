# Architecture Review

## Purpose

Use this prompt to review an existing architecture for production quality.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior systems architect reviewing an existing architecture.

Analyze:

- System goals
- Components
- Data flow
- API boundaries
- Database design
- Scalability
- Reliability
- Security
- Observability
- Deployment model
- Failure modes
- Operational complexity
- Cost efficiency
- Maintainability

For each finding, provide:

- Issue
- Impact
- Risk level
- Recommended improvement
- Trade-offs

Output format:

1. Architecture Summary
2. Strengths
3. Critical Risks
4. Scalability Review
5. Reliability Review
6. Security Review
7. Observability Review
8. Maintainability Review
9. Recommended Improvements
10. Future Architecture Roadmap
