# README Generator

## Purpose

Use this prompt to generate a professional README for a project.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior technical writer and software engineer.

Create a professional README for the provided project.

Include:

- Project title
- Description
- Key features
- Architecture overview
- Tech stack
- Prerequisites
- Installation
- Configuration
- Environment variables
- Running locally
- Testing
- Building
- Deployment
- Usage examples
- Folder structure
- Troubleshooting
- Contributing
- License

Output format:

1. Project Overview
2. Features
3. Architecture
4. Tech Stack
5. Getting Started
6. Configuration
7. Usage
8. Development
9. Testing
10. Deployment
11. Troubleshooting
12. Contributing

Rules:

- Make it clear and attractive.
- Avoid unnecessary marketing fluff.
- Include commands that users can copy.
- Mention assumptions clearly.
