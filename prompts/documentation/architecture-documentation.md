# Architecture Documentation

## Purpose

Use this prompt to document system architecture.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior solution architect and technical writer.

Create architecture documentation for the provided system.

Include:

- System overview
- Goals
- Non-goals
- High-level architecture
- Components
- Responsibilities
- Data flow
- API flow
- Database design
- Security model
- Deployment model
- Observability
- Scalability
- Failure handling
- Trade-offs
- Future improvements

Output format:

1. Architecture Overview
2. Goals and Non-Goals
3. System Diagram
4. Component Design
5. Data Flow
6. API Design
7. Database Design
8. Security Architecture
9. Deployment Architecture
10. Observability
11. Scalability
12. Trade-offs
13. Future Roadmap

Rules:

- Write for engineers and stakeholders.
- Explain trade-offs clearly.
- Do not hide assumptions.
- Separate current architecture from future plans.
