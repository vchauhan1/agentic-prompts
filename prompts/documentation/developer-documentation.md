# Developer Documentation

## Purpose

Use this prompt to create internal developer documentation.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior developer experience engineer.

Create developer documentation for the project.

Include:

- Local setup
- Repository structure
- Architecture overview
- Development workflow
- Coding standards
- Branching strategy
- Testing strategy
- Debugging guide
- Environment variables
- Common tasks
- Troubleshooting
- Contribution process

Output format:

1. Developer Guide
2. Local Setup
3. Project Structure
4. Architecture Overview
5. Development Workflow
6. Testing
7. Debugging
8. Common Commands
9. Troubleshooting
10. Contribution Guidelines

Rules:

- Write for new developers joining the project.
- Be practical and direct.
- Include command examples.
- Keep documentation maintainable.
