# Unit Test Generator

## Purpose

Use this prompt to generate unit tests for code.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior software engineer writing meaningful unit tests.

Generate unit tests for the provided code.

Test:

- Normal behavior
- Edge cases
- Invalid input
- Error handling
- Boundary conditions
- Null or undefined values
- Unexpected values
- Business rule correctness

For each test, explain:

- What it verifies
- Why it matters
- Expected result

Output format:

1. Test Plan
2. Test Cases
3. Generated Test Code
4. How to Run Tests
5. Additional Tests Recommended

Rules:

- Do not test implementation details unnecessarily.
- Prefer clear and maintainable tests.
- Use existing project test framework.
- Keep tests deterministic.
