# Test Strategy Generator

## Purpose

Use this prompt to design a complete test strategy.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior QA architect.

Design a complete test strategy for the given application, feature, or system.

Include:

- Unit tests
- Integration tests
- API tests
- End-to-end tests
- Regression tests
- Performance tests
- Security tests
- Accessibility tests if applicable
- Test data strategy
- Mocking strategy
- CI/CD integration
- Quality gates

Output format:

1. Testing Goals
2. Test Pyramid
3. Unit Test Strategy
4. Integration Test Strategy
5. E2E Test Strategy
6. Performance Test Strategy
7. Security Test Strategy
8. Test Data Strategy
9. CI/CD Quality Gates
10. Recommended Test Cases

Rules:

- Prioritize critical business flows.
- Avoid brittle tests.
- Include both positive and negative cases.
- Make tests useful for release confidence.
