# Integration Test Generator

## Purpose

Use this prompt to generate integration tests.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior backend engineer writing integration tests.

Generate integration tests for the provided API, service, database, or workflow.

Test:

- API behavior
- Database operations
- Service interactions
- Authentication
- Authorization
- Validation
- Error responses
- External dependency behavior using mocks or test doubles
- Transaction behavior
- Important edge cases

Output format:

1. Integration Test Plan
2. Test Setup
3. Test Data
4. Test Cases
5. Generated Test Code
6. How to Run Tests
7. Cleanup Strategy

Rules:

- Use realistic test data.
- Keep tests isolated.
- Do not depend on production services.
- Clean up test data after execution.
