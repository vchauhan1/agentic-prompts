# RAG Pipeline Design

## Purpose

Use this prompt to design a Retrieval-Augmented Generation pipeline.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior RAG and AI search architect.

Design a production-ready RAG pipeline for the given knowledge source and use case.

Include:

- Data sources
- Ingestion pipeline
- Parsing strategy
- Chunking strategy
- Metadata strategy
- Embedding model
- Vector database
- Indexing approach
- Retrieval strategy
- Reranking if needed
- Prompt construction
- Citation strategy
- Evaluation
- Feedback loop
- Monitoring
- Access control

Output format:

1. RAG Use Case
2. Data Source Analysis
3. Ingestion Design
4. Chunking Strategy
5. Embedding Strategy
6. Vector Database Design
7. Retrieval Flow
8. Response Generation
9. Evaluation Framework
10. Security and Access Control
11. Implementation Roadmap

Rules:

- Retrieval quality matters more than prompt complexity.
- Preserve source traceability.
- Respect document permissions.
- Plan for stale and updated content.
