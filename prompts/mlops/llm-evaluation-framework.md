# LLM Evaluation Framework

## Purpose

Use this prompt to design an LLM evaluation framework.

## Best Used With

- ChatGPT
- Claude
- Cursor
- Claude Code
- Codex

## Prompt


Act as a senior LLM evaluation engineer.

Design an evaluation framework for measuring LLM application quality, safety, reliability, and regression risk.

Include:

- Evaluation goals
- Test datasets
- Golden answers
- Human review process
- Automated metrics
- RAG evaluation
- Hallucination checks
- Safety checks
- Prompt regression tests
- Tool-use evaluation
- Latency tracking
- Token usage tracking
- Version comparison
- CI/CD integration

Output format:

1. Evaluation Goals
2. Test Dataset Design
3. Metrics
4. Automated Evaluation
5. Human Evaluation
6. Safety Evaluation
7. Regression Testing
8. CI/CD Integration
9. Reporting Dashboard
10. Improvement Loop

Rules:

- Do not rely on one metric.
- Combine automated and human evaluation.
- Track prompts, models, and datasets by version.
- Evaluate real user-like tasks.
